This tool uncaps your Korone FPS.
Originally made by `slushify.` on Discord

## ✨ Features
- Unlock FPS beyond the default limit  
- Simple and lightweight

## 🚀 Installation
1. Download the latest release from the [Releases](../../releases) page.  
2. Extract the files to a folder of your choice.  
3. Run `Korone-FPS-Unlocker.exe`.  
4. Launch Korone and enjoy unlimited FPS!  

## 📁 Building
1. Clone the repository and open the solution in **Visual Studio 2022**.
2. Select the **configuration**:  `Release | x64`.
